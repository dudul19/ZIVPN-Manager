### Features
Some of this script's features:
- Automatic Installation
- Interactive Menu
- Complete User Management
- Automatic Cleanup
- Instant Removal
- Uninstaller

#### Requirement
- A Linux based System ```VPS```
  - Ubuntu 18+ VPS with a public IP.
- Root or sudo access for installation and configuration.

#### Installation
Follow these steps to install and set up ZiVPN Manager:

#### AMD64 (x86_64)
```
sudo wget -O /usr/local/bin/zi.sh https://raw.githubusercontent.com/dudul19/udp-zivpn/main/zi.sh && sudo chmod +x /usr/local/bin/zi.sh && sudo zi.sh
```

### ARM64
```
sudo wget -O /usr/local/bin/zi2.sh https://raw.githubusercontent.com/dudul19/udp-zivpn/main/zi2.sh && sudo chmod +x /usr/local/bin/zi2.sh && sudo zi2.sh
```

#### Usage

Here's how to use ZiVPN Manager:

> After the installation is complete, you can access the management menu by running the following command:

```
zivpn
```
> From this menu, you can manage all your Zivpn service.


#### Uninstall

If you want to remove Zivpn from your server, you can do so through the menu or by running the following command directly:

```bash
uninstall.sh
```
